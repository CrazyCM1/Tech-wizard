How it works

Simple workflow. Paste a job description, paste resumes, and get ranked results instantly.

1

Paste Job Description

The system detects required skills, preferred skills, and important keywords.

2

Paste / Upload Resumes

It extracts skills, tools, experience signals, and builds a structured profile.

3

Get Ranked Candidates

Candidates are scored and ranked. Each result includes a clear explanation report.

4

Export the Report

Download a text report or print the report as PDF directly from the browser.

About this project

SkillMatch is a browser-based Resume Screening & Matching system built using vanilla HTML, CSS and JavaScript. It helps reduce manual screening effort, improves candidate-job matching, and provides transparent evaluation.
